Supported Conversions & Oracle Reference

Data Types

  • VARCHAR / TEXT → VARCHAR2 (max 4000 bytes)
  • CHAR vs VARCHAR2 — CHAR is fixed-length, padded with spaces
  • INT / INTEGER → NUMBER(38)
  • BOOLEAN → NUMBER(1) — Oracle has no native BOOLEAN in SQL
  • DATETIME → DATE (includes time) / TIMESTAMP
  • NVARCHAR2 for Unicode; CLOB / BLOB for large objects
  • AUTO_INCREMENT → SEQUENCE + DEFAULT or IDENTITY column (12c+)

Functions

  • IFNULL / COALESCE → NVL(expr, default)
  • NVL2(expr, val_if_not_null, val_if_null)
  • DECODE(expr, search, result [, default])
  • NULLIF(expr1, expr2) → NULL if equal
  • NOW() → SYSDATE / SYSTIMESTAMP
  • CONCAT → || operator
  • LIMIT / TOP → ROWNUM / FETCH FIRST n ROWS ONLY (12c+)
  • GROUP_CONCAT → LISTAGG(col, ',') WITHIN GROUP (ORDER BY col)
  • TO_DATE('2024-01-15', 'YYYY-MM-DD') — always specify format mask

Syntax

  • ILIKE → UPPER(col) LIKE UPPER(:val)
  • EXCEPT → MINUS
  • Backtick identifiers → double-quoted
  • MERGE INTO target USING source ON … WHEN MATCHED … WHEN NOT MATCHED …
  • CONNECT BY PRIOR — hierarchical queries (no recursive CTE needed)
  • PIVOT / UNPIVOT — native Oracle syntax
  • String escaping: use q'[…]' for strings containing quotes
  • JOIN syntax normalization — ANSI JOIN preferred over (+) syntax

Source Dialects

  • MySQL / MariaDB
  • PostgreSQL
  • SQL Server (T-SQL)
  • BigQuery, Snowflake, Redshift
  • SQLite, Hive, Spark, Presto, Trino, DuckDB

Quick Tips

  • Oracle requires FROM DUAL for SELECT without a table: SELECT 1 FROM DUAL
  • String comparison is case-sensitive — use UPPER() or NLS_SORT
  • Sequences: CREATE SEQUENCE s START WITH 1 INCREMENT BY 1; then s.NEXTVAL
  • Empty string '' = NULL in Oracle — use IS NULL to check
  • Date arithmetic: SYSDATE + 1 adds one day; SYSDATE - 1/24 subtracts one hour
  • Use DBMS_METADATA.GET_DDL to extract object definitions
