Supported Conversions

Data Types

  • VARCHAR / TEXT → VARCHAR2
  • INT / INTEGER → NUMBER
  • BOOLEAN → NUMBER(1)
  • DATETIME → DATE / TIMESTAMP
  • AUTO_INCREMENT → SEQUENCE pattern

Functions

  • IFNULL / COALESCE → NVL
  • NOW() → SYSDATE
  • CONCAT → || operator
  • LIMIT / TOP → ROWNUM / FETCH FIRST
  • GROUP_CONCAT → LISTAGG

Syntax

  • ILIKE → UPPER/LOWER LIKE
  • EXCEPT → MINUS
  • Backtick identifiers → double-quoted
  • String escaping differences
  • JOIN syntax normalization

Source Dialects

  • MySQL / MariaDB
  • PostgreSQL
  • SQL Server (T-SQL)
  • BigQuery, Snowflake, Redshift
  • SQLite, Hive, Spark, Presto, Trino
